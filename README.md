# 🍽️ Recipe App (Flutter)

A beautiful and simple **Recipe App built with Flutter** that allows users to browse delicious recipes with a clean and modern UI. This project is designed for learning and showcasing Flutter development skills.

## 🚀 Features

- 🍲 Display list of recipes  
- 📖 Detailed recipe view  
- 🖼️ Clean and modern UI  
- ⚡ Smooth performance  
- 📱 Responsive design for all devices  
- 🎯 Beginner-friendly Flutter structure

- Flutter 💙  
- Dart 🎯  
- Material Design UI
